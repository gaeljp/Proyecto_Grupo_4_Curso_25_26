# Proyecto_Grupo_4_Curso_25_26

Repositorio del proyecto del modulo proyecto intermodular del grupo 4 para el curso 2025-2026.

## Descripción

Este repositorio contiene la memoria, diagramas y recursos de interfaz de una aplicación de gestión de almacenaje doméstico.  
La aplicación permite organizar objetos y productos del hogar mediante una estructura jerárquica formada por habitación, mueble y ubicación concreta.

## Contenido del repositorio

- `memoria.pdf`: memoria actualizada del proyecto.
- `presentacion.txt`: enlace a la presentación del trabajo.
- `diagramas/`: diagramas UML, entidad-relación y otros esquemas del proyecto.
- `ui/`: diseños de las pantallas principales de la aplicación.

## Funcionalidades del proyecto

- Gestión de objetos del hogar.
- Organización por habitaciones, muebles y ubicaciones concretas.
- Consulta, alta, modificación y eliminación de artículos.
- Gestión de imágenes asociadas a los objetos.
- Notificaciones y avisos.
- Exportación de datos.
- Estadísticas y sugerencias de orden.
- Sincronización entre varios usuarios del hogar.
